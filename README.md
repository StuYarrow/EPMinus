# EPMinus

This is a fork of EPPlus v4.5.3.3, the last free version of EPPlus.  It targets .NET 10 and the dependencies are updated, but it is otherwise essentially unmodified. Much of the functionality is untested.

**If you want the latest and best EPPlus, go and get the real thing from the original authors: https://github.com/EPPlusSoftware/EPPlus.**
